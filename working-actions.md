# Working Web Actions

Confirmed `/web_action` commands that work with auto-webmcp. Requires Chrome with the auto-webmcp extension installed.

---

## GitHub

### Create a public repository

```
/web_action create a public github repository called my_mcp_test.
```

### Create an issue

```
/web_action create a new issue on github in repo my_mcp_test titled "test issue 2". my username is prasanna-gyde
```

---

## Bitbucket

### Create a private repository

/web_action create a private bitbucket repository called my_mcp_test.

## Gmail

### Compose and send an email

```

```

## Salesforce

> Note: Include your Salesforce org URL in the prompt, or have the org open in a tab before running the action.

### Create an Opportunity

```
/web_action create a new opportunity in Salesforce named "Lakme Deal" with amount $52000 my salesforce org is on https://d0o000001aqeleas-dev-ed.lightning.force.com
```

### Create a Contact

```
/web_action Create a new Contact in Salesforce. Set First Name to "Jane", Last Name to "Smith", Title to "VP of Engineering", Phone to "415-555-0100", Email to "jane.smith@example.com", and Account Name to "Acme Corp"
```

### Create a Case

```
/web_action Create a new Case in Salesforce. Set the Subject to "Login issue reported by customer", Status to "New", Priority to "High", and Origin to "Phone". Add a description "Customer cannot log in after password reset."
```

### Create a Lead

```
/web_action Create a new Lead in Salesforce. Set First Name to "Alex", Last Name to "Johnson", Company to "TechCorp", Title to "CTO", Email to "alex.johnson@techcorp.com", Phone to "415-555-0200", and Lead Source to "Web".
```
