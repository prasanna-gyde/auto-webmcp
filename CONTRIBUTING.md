# Contributing to auto-webmcp

## Prerequisites
- Node.js 18+
- npm 9+
- Chromium (installed automatically by Playwright)

## Setup
```sh
npm install
npx playwright install chromium
```

## Build
```sh
npm run build        # compile src/ → dist/
npm run build:watch  # rebuild on file change
npm run typecheck    # type-check without emitting
```

## Test
```sh
npm test             # run Playwright tests (headless Chromium)
```

Tests live in `tests/analyzer.spec.ts` and use HTML fixtures in `tests/fixtures/`.

## Project layout
```
src/
  config.ts       — Config types + resolveConfig()
  schema.ts       — HTML input → JSON Schema mapping
  analyzer.ts     — Infers tool name/description/schema from a form
  discovery.ts    — MutationObserver + DOMContentLoaded scanner
  registry.ts     — navigator.modelContext wrapper
  interceptor.ts  — Fills form fields + intercepts submit events
  enhancer.ts     — Optional LLM-based description enrichment
  index.ts        — Public autoWebMCP() API + IIFE auto-init
dist/             — Built output (committed; generated by build.mjs)
tests/            — Playwright test suite
```

## Submitting changes
1. Fork the repo and create a feature branch.
2. Make your changes and ensure `npm test` passes.
3. Open a pull request with a clear description of what and why.

Please keep PRs focused — one feature or fix per PR.
