About This Page
Pure inference mode. None of these forms use toolname or tooldescription attributes. Tool names and schemas are inferred entirely from the form's id, field names, and HTML structure.

Password and file inputs are automatically excluded from all schemas. Toggle "Tool JSON" on any form to see what was inferred.
